# Bella's Cafe Website

A responsive website for a local cafe with bougainvillea flower theme.

## 📁 Files
- `index.html` - Main website
- `style.css` - All styling
- `script.js` - Interactive features

## 🚀 How to Run
1. Download all files
2. Open `index.html` in browser
3. Or use Live Server in VS Code

## ✨ Features
- Responsive design (works on phone, tablet, desktop)
- Interactive menu with filtering
- Shopping cart
- Reservation form
- Newsletter signup
- Photo gallery

## 🛠️ Technologies
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

## 🎯 Business Impact
- Mobile-friendly = more customers
- Online cart = higher sales
- Reservations = less phone calls
- Newsletter = repeat customers
- Beautiful design = free social media promotion

## 📱 Responsive
Works on all screen sizes:
- Desktop (>768px)
- Tablet (768px)
- Mobile (<768px)
